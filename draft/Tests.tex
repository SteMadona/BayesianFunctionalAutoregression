\section{Studies with synthetic data and simulations}
All the data were generated using equation \ref{generazione}. \\
The number of observations per curve is $n_i = n = 200, \forall i \in \{1, ... , T\}$. A specific grid of equally spaced points for each function was defined without loss of generality, from which the respective functional data were generated.
\subsection{First Simulation: no autoregressive component}
In the first simulation the data were generated by a linear comibination of B-splines without the autoregressive component. The $\Phi$ matrix is generated from an Inverse Wishart with moderate variance and diagonal scale matrix.  \\
The arbitrary selected vector of true coefficients is $\boldsymbol{\alpha} = \{ -2 , 0.5, 1, 0, 1.5, -1, 0, 1.5, -1.5, -1, 1, 0 \}$, so that: 
\begin{gather*}
    f_t(x_i) = \sum_{j=1}^k \alpha_j B_j(x) + \epsilon_t(x_i) \\
    =(-2 , 0.5, 1, 0, 1.5, -1, 0, 1.5, -1.5, -1, 1, 0)' 
    \left(
    \begin{matrix}
        B_1(x_i) \\
        B_2(x_i) \\
        \vdots \\
        B_k(x_i) 
    \end{matrix} \right) + \epsilon_t(x_i), \quad \text{with } \quad \epsilon_t(x_i) \sim N(0, \sigma^2 \phi_i)
\end{gather*}
where $(B_1(x_i), ..., B_k(x_i)'$ is the vector with the B-splines evaluated at $x_i$ used to generate the $i$th point of the synthetic curves.
