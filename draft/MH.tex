\section{Metropolis-Hastings for $\Phi$}
Instead of trying to explicit $\Phi$ in a conjugate way, we try the approach Matropolis-within-Gibbs; in particular, at every iteration of the Gibbs sampler, $\Phi$ is being generated by a Metropolis-Hastings iteration. \\
We modify the prior of $\Phi$ such that: 
\begin{equation}
    \Phi \sim IW(\Psi_0, \nu_0)
\end{equation}
At every iteration, we generate a value from the proposal distribution
\begin{equation}
    q(\Phi^* | \Phi_{t-1}) \sim IW(\Phi_{t-1}, \nu_0)
\end{equation}
We then compute the logarithm of acceptance probability as: 
\begin{gather*}
    \alpha(\Phi^*, \Phi_{t-1}) = \min \left \{1, \exp\{\log L(f_1, ..., f_T | \Phi^*) + \log\pi(\Phi^*) + \log q(\Phi_{t-1} | \Phi^*) + \\
    - \log L(f_1, ..., f_T | \Phi_{t-1}) - \pi(\Phi_{t-1}) - q(\Phi^* | \Phi_{t-1})  \} \right\}
\end{gather*}

\subsection{Multiple Try Metropolis Hastings}
The previous method is correct from a theoretical point of view, but suffers some numerical problems due to the IW density function in many dimensions being almost $0$ everywhere. \\
To overcome this issue, the first attempt is to use a method called \textit{Multiple Try Metropolis Hastings (MTMH)}. This method, at every iteration, follows this algorithm: 
Suppose the current state is $\Phi_{t-1}$.
\begin{enumerate}
    \item Draw $\Phi_1^*, ..., \Phi_m^*$ from $q(\Phi_{t-1}, ^.)$. Compute the weights as:
    \begin{equation}
        w(\Phi_j^*, \Phi_{t-1}) = L(f_1, ..., f_T | \Phi_j^*)\pi(\Phi_j^*)q(\Phi_{t-1} | \Phi_j^*)
    \end{equation}
    \item Select $\Phi^*$ from $\Phi^*_j$ with probability proportional to the weights. 
    \item Produce a reference set by drawing $\Phi_1, ..., \Phi_{m-1}$ from distribution $q(\Phi^*, ^.)$. Set $\Phi_m = \Phi_{t-1}$ (the current point). 
    \item Accept $\Phi^*$ with probability: 
    \begin{equation}
        \alpha(\Phi^*, \Phi_{t-1}) = \min \left \{1, \frac{w(\Phi_1^*, \Phi_{t-1}) + ... + w(\Phi_m^*, \Phi_{t-1})}{w(\Phi_1, \Phi^*) + ... + w(\Phi_m, \Phi^*)} \right\}
    \end{equation}
\end{enumerate}
Actually, the density problem still holds, so we have to try something different.


\subsection{Fourier Decomposition of Error Term}
We have that 
\begin{equation}
    \Phi = Cov(\epsilon_t(x), \epsilon_t(x'))
\end{equation}
but with a Metropolis Hastings algorithm and a very fine $x$ grid, the posterior for $\Phi$ becomes computationally untractable. We the choose an orthonormal Fourier basis system $\{d_k(x)\}_{k=1}^K$ in order to approximate:
\begin{equation}
    \epsilon_t(x) \approx \sum_{k=1}^K r_{t, k}d_k(x)
\end{equation}
which, in a vectorial representation becomes $\epsilon_t(x) = D(x)\textbf{r}_t$. \\
We also assume that $r_t \sim N(0, \Psi)$. The covariance between points $x$ and $x'$ becomes
\begin{equation}
    Cov(\epsilon_t(x), \epsilon_t(x')) = Cov(D(x)\textbf{r}_t, D(x')\textbf{r}_t)) = D(x) \Psi D(x')^T
\end{equation}
